❓Implement an algorithm to subtract two strings, represented as linked lists, in the order they appear and return the result as a linked list.The linked lists may contain digits from 0 to 9. You can assume that the input strings are well-formed and do not contain leading zeros, except for the number zero itself.For example:l1 = ["1","2","3"]l2 = ["9","2"]The expected output is a linked list representing the subtraction of the two input strings:result = ["-", "8", "1"]The algorithm should handle the case where the result of the subtraction is negative. In this case, the first element of the result linked list should be "-" to indicate a negative number.To solve this problem, follow these steps:1. Initialize two pointers, one for each linked list, pointing to the last node.2. Initialize a carry variable to 0.3. Iterate through the linked lists until both pointers reach the beginning of the lists.4. Subtract the corresponding digits of the two linked lists and add the carry.5. If the result is negative, add 10 to the result and set the carry to -1.6. Create a new node with the result digit and add it to the result linked list.7. Move the pointers to the previous nodes.8. After the loop ends, check if there is still a carry remaining.9. If there is a carry, create a new node with the carry value and add it to the result linked list.10. Reverse the result linked list to obtain the correct order.11. Return the result linked list.This modified problem retains the same steps as the original problem, but requires the addition of more complex logic to handle negative results. Additionally, the problem assumes that the input strings are well-formed and do not contain leading zeros.
